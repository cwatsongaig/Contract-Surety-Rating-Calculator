# Changelog

## v1.0.0 (Build 1) — 2026-05-08

### Features
- Rate Lookup: Select rate rows and copy as a branded image (GAI logo) for email
- Premium Calculator: Copy calculated premium breakdown as a branded image
- Compare Plans: Copy side-by-side plan comparison as a branded image
- Company defaults to "Great American Insurance" across all screens
- Plan Comparison uses clickable buttons for plan selection

### Initial App Capabilities
- Rate Lookup with filterable rate table
- Premium Calculator with tiered rate breakdown, debit/credit, time surcharge, maintenance bonds
- Commission scale viewer
- Plan Comparison with side-by-side rates and premium totals
